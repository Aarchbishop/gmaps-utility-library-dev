DragZoom Back Button Example with Map Types

Illustrating the the saving and restoring of map context including the map type using the DragZoom Back Button.
  1) The map shows Greater New York. The DragZoom Button is in the top left of the map and the Map Type Controls are in the upper right of the map.
  2) Choose an area using the DragZoom around New York Harbor (about 1/3 of the screen).
  3) Change the Map Type to Hybrid.
  4) Do another DragZoom highlighting the Brooklyn Waterfront.
  5) Change the Mapt Type to Satellite.
  6) Use the DragZoom Back Button to reverse the sequence. The Map Types will be restored to the value for each zoom level.
  Play around, zooming in and out using the DragZoom and changing the Map Type Controls.
