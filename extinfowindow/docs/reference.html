class ExtInfoWindow

This class is used to create a custom GInfoWindow that will mimic the behavior of the standard GInfoWindow. Additionally this class lets you easily theme your GInfoWindow through CSS id attributes and even pull content from a supplied Ajax url in the constructor, giving the user a more control over every aspect of the GInfoWindow.

Constructor

Constructor Description
ExtInfoWindow(marker, windowId, html, opt_opts) Creates a new ExtInfoWindow with content and styling based on the passed parameters.

Methods

Methods Return Value Description
resize() none Recalculate the height of the window
repositionMap() none Determines the location of the GInfoWindow's corners in relation to the viewable map region in order to pan the map, if need be, so that the entire GInfoWindow is always displayed.

class ExtInfoWindowOptions

This class represents optional arguments to the ExtInfoWindow constructor. It has no constructor, but is instantiated as object literal.

Properties

Properties Type Description
ajaxUrl String A string of the relative url where the ExtInfoWindow should request HTML data. NOTE: content returned is assumed to be valid HTML content.
beakOffset Number The number of pixels a the beak, or pointer end, of the ExtInfoWindow should be moved up vertically. This is useful when working with an ExtInfoWindow with a border. The default value is 0.
paddingX Number The number, in pixels, that ExtInfoWindow should pad on the left and right sides when positioning itself inside the viewable map region. The default value is 0.
paddingY Number The number, in pixels, that ExtInfoWindow should pad on the top and bottom sides when positioning itself inside the viewable map region. The default value is 0.
